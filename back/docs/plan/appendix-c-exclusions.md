# 부록 C: 명시적 제외 & 알려진 간극

> 상위: [plan.md](../plan.md)

## C.1 명시적 제외

구현하지 않기로 결정한 영역:

- 비전투 NPC AI 자율 스케줄
- 아이템 내구도 / 파손
- 날씨·환경의 자동 이벤트
- 파티(파티원 선택·해산) 시스템 — §11.8 companions 가 동일 요구를 pocket-monster 방식으로 충족
- **설계자(Design) 에이전트**: `src/agents/design/creator.md` (레거시) 는 엔티티 저작 가이드로, 플레이 런타임에 포함되지 않는다. 디자이너가 신규 아이템·NPC·장소·퀘스트를 만들 때 참고하는 밸런스 테이블·체크리스트. back 에서는 `config/profiles/{name}/` 에 JSON 시드로만 반영. 저작 에이전트 자체는 당분간 생략.

## C.2 알려진 간극 (미구현이지만 열려 있음)

- 스킬 사용 중 사정거리 외 자동 재배치
- NPC ↔ NPC 상거래
- 잠긴 문 / 상자의 물리적 파괴 경로 (현재는 열쇠 / lockpick roll 만)
- 킬 카운트 같은 누적 퀘스트 조건
